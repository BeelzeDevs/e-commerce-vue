# Backend: .NET 8 API con Entity Framework y PostgreSQL
# Frontend: Vue 3 + TypeScript + Tailwind


## Backend: 
## - .Net 8.0 framework Web API
## - Npgsql package Entity Framework Core PostgreSQL
## - AspNetCore Authentication JwtBearer 8.0.4

## Frontend:
## - Vue 3
## - TypeScript
## - Vue Router
## - Pinia
## - Tailwind CSS
## - Autoprefixer (compability with navs) y Postcss (execute css plugins)
